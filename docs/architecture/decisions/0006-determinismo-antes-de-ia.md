# ADR 0006 — Determinismo antes de IA generativa

**Status:** Aceito

## Decisão
MVP 100% determinístico. Arquitetura preparada para enriquecimento por LLM no futuro.

## Justificativa
- Consistência e profissionalismo (clientes equivalentes recebem devolutivas equivalentes).
- Auditabilidade rastreável.
- Custo operacional zero.
- Curadoria humana como ativo intelectual da Ferzion.

> IA deve potencializar o produto. Não sustentar o produto.

## Consequências
**Positivas:** MVP confiável e previsível; fundação sólida; custo desprezível.
**Negativas:** maior esforço inicial de redação (compensado pelo valor curatorial).
